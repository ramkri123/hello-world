svlearn.compute.es_indexer_job.call_rest_partition(records)[source]

This method is called for each partition of the incoming dataframe. For each partition, it does the following:

  1. Create a connection to the ElasticSearch instance.

  2. For each row in the partition, it calls the ElasticSearch REST endpoint

    to index the text.

  3. Update the CHUNK table to set the ES_INDEXED column to True.
